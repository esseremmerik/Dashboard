== Description ==

Description:  *Select Shortcode-post-display or/and Sidebar Widget. Now with administration panel* - Collection of over 31.000 Games for In-Post and Pages or the Sidebar with jQuery support included. The Game Widget opens in a Lightbox. The In-Post-display shows the arcade in Pages using shortcode. The Online Games Collection enables you to delivery visitors Games without the need of a resource-consumpting arcade system. A 411 Games widget has been added to expand the functionality of this plugin. The Webmaster decides what he wants to display.
The Games are auto updated. Tutorial videos in the backend.

You get:
- a "Set and forget" Games page with over 31.000 Games
- plugin and sidebar widgets
- "Stay on your Site" - concept.


Demo / Example: [Wordpress Themes](http://www.wordpressthemes247.com)
Developer and Overview: [Free Online Games](http://www.freeonlinegamesplay247.com) Home


== Installation ==
= To Install: =

1. Upload `online-games.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the Instructions on the administration panel
